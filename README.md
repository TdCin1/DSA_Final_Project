### Text Interface
Run using `--tui`.
![](https://i.imgur.com/M05QvUj.png)

## Building
### Windows
SDL2, SDL2 Image, and SDL2 TTF must be installed and configured in CMake (if not automatically).

### Linux
Install SDL2, then build using CMake:
```
mkdir build && cd build
cmake ..
make
```
